# Sprint_5 — UI автотесты Stellar Burgers

Проект содержит автотесты на Selenium + Pytest для сайта Stellar Burgers.

## Что проверяется
- Регистрация (успех и ошибка короткого пароля)
- Вход (4 способа: с главной, через личный кабинет, через форму регистрации, через восстановление пароля)
- Переход в личный кабинет
- Переход из личного кабинета в конструктор (по кнопке и по логотипу)
- Выход из аккаунта
- Вкладки конструктора: «Булки», «Соусы», «Начинки» (проверка активной вкладки по классу)
