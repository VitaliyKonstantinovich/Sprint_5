# UI-тесты для проекта Stellar Burgers (Sprint 5)

В этом проекте написаны UI-автотесты для сайта Stellar Burgers с использованием Python, Pytest и Selenium.

### Структура проекта:
*   `tests/` — папка со всеми тестами, разбитыми по функционалу (регистрация, логин, переходы, конструктор).
*   `conftest.py` — файл с фикстурой для запуска и закрытия браузера Chrome.
*   `locators.py` — файл, в котором собраны все XPath-локаторы элементов сайта.
*   `helpers.py` — функции-помощники (генерация уникального email и паролей для тестов).

### Что покрыто тестами:
1.  **Регистрация:** успешная регистрация и ошибка при пароле меньше 6 символов.
2.  **Вход в аккаунт:** 4 разных способа (кнопка на главной, кнопка "Личный кабинет", ссылки в формах регистрации и восстановления пароля).
3.  **Переходы:** переход в личный кабинет, переходы из личного кабинета в конструктор.
4.  **Выход:** выход из аккаунта.
5.  **Конструктор:** переходы по вкладкам "Булки", "Соусы", "Начинки".

### Как запустить тесты:
1. Установите зависимости: `pip install pytest selenium webdriver-manager`
2. Запустите тесты командой: `pytest -v`
